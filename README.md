# Dotfiles

Set up Mac:

```bash
wget -qO- https://raw.githubusercontent.com/peterszarvas94/dots/master/setup_mac | bash
```

Set up Omarchy:

```bash
yay -S --noconfirm --needed wget
wget -qO- https://raw.githubusercontent.com/peterszarvas94/dots/master/setup_omarchy | bash
```
