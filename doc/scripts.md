# Scripts package (`stow/scripts`)

## Owns

- `~/.local/bin`

## Purpose

Small local CLI tools used by shell, tmux, Hyprland, and Waybar workflows.

## High-usage scripts

- `stow/scripts/.local/bin/theme`: toggles desktop dark/light preference with `gsettings`.
- `stow/scripts/.local/bin/tmux-sessionizer`: fuzzy-select directory and switch/create tmux session.
- `stow/scripts/.local/bin/branch`: fuzzy git branch checkout helper.
- `stow/scripts/.local/bin/omarchy-cmd-screenrecord`: recording workflow with Waybar signal support and optional post hook.
